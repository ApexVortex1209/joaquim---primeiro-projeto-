# Joaquim - Primeiro Projeto 🚀

Meu primeiro projeto web desenvolvido para praticar conceitos fundamentais de desenvolvimento front-end e versionamento de código com Git e GitHub.

## 📖 Sobre o Projeto

Este projeto foi criado com o objetivo de aplicar conhecimentos iniciais em desenvolvimento web, focando na construção de interfaces e organização de código.

## ✨ Funcionalidades

- Estrutura básica de uma página web
- Layout organizado
- Prática de HTML, CSS e JavaScript
- Versionamento utilizando Git e GitHub

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/ApexVortex1209/joaquim---primeiro-projeto-.git
```

2. Entre na pasta do projeto:

```bash
cd joaquim---primeiro-projeto-
```

3. Abra o arquivo `index.html` em seu navegador.

## 🎯 Objetivo

Este projeto marca o início da minha jornada como desenvolvedor, servindo como base para aprender boas práticas de programação, desenvolvimento web e colaboração com GitHub.
